# Portfólio Pessoal - Trabalho da disciplina Programação para Internet I - IFPI

# Integrantes
* Eli Ruan dos Santos Carvalho

# Tema
Analista de Dados / Estudante de análise e desenvolvimento de sistemas.

# Breve Descrição do Portfólio
Este projeto consiste em um portfólio web responsivo desenvolvido somente com HTML e CSS puro, simulando a página profissional de um Analista de Dados/Sistemas. O site foi estruturado para apresentar competências técnicas em arquitetura de dados (SQL Server, Oracle), ferramentas analíticas (Looker Studio, DBeaver, Powerr BI) e em linguagens de programação com foco em análise de dados. A página exibe seções de texto "Sobre Mim", "Habilidades" em formato de tabela, "Projetos" e um formulário de contato.

# Principais Desafios Encontrados
* Fazer a estruturação de modo responsivo;
* Fazer as formatações adequadas em CSS.
* Fazer os ajustes das imagens, também pra deixar Responsivas, e manter a proporção da página.
